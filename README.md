# NCVFirstComment
コメント番号１を獲得したリスナーさんを記録するプラグインです

## Requirement
.NET Framework 4.6.2

## Installation
NCV本体と同階層にあるpluginsフォルダ内へコピーしてください

## Note
過去に開発・配布していた「1CommentReply」の後継プラグインです
